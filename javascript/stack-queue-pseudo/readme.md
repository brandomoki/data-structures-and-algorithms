# stack-queue-psuedo

## Challenge

- Implement a Queue using two Stacks.
  - Methods:
    - enqueue
      - Arguments: value
      - Inserts a value into the PseudoQueue, using a first-in, first-out approach.
  - dequeue
    - Arguments: none
    - Extracts a value from the PseudoQueue, using a first-in, first-out approach.

## Approach & Efficiency

- O(1)
